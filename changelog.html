Version 1.2.0

Features

  • Adds in a deserializer system (ported from Crafting Harmonics) to help in converting data from JavaScript code into Java classes.
  • Cleaning up logging a bit on the internal side of things; more things will use your logger instead of STDERR. Provided more exceptions for things.
  • Adding function to stringify JSON objects; these must be a JSObject (generally from JavaScript code) or they will just be undefined.
